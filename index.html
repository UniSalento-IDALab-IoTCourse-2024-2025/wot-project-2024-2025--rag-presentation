💬 Sistemi RAG per un agente intelligente in ambito universitario

Questo progetto nasce con l’obiettivo di facilitare la consultazione delle informazioni universitarie attraverso un chatbot intelligente basato su tecnologie Retrieval-Augmented Generation (RAG).

🔧 Tecnologie principali

  • LangChain per orchestrare il flusso LLM + retrieval
  • Cohere come LLM multilingua
  • FAISS per il database vettoriale
  • MongoDB + Node.js per la gestione utenti e sessioni
  • React Native per l’interfaccia mobile

📦 Repository del progetto

  • 📁 Presentazione generale
  • 🧠 RAG Core (Python)
  • 🛠️ UserChat Service (Node.js)
  • 📱 Frontend (React Native)
